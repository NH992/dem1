# AI计算器（脱裤子放屁版）

这是一个诙谐幽默的计算器应用，它并不是真正进行计算，而是通过调用Windows系统自带的计算器来完成计算任务，然后将结果显示在自己的界面上。

## 功能特点

- 用户在本应用界面输入数学表达式
- 应用启动系统计算器
- 自动在系统计算器中输入相同的表达式
- 读取系统计算器的计算结果
- 将结果显示回本应用界面

## 技术实现

- 使用PyQt5创建图形用户界面
- 使用pyautogui控制键盘输入
- 使用pywin32管理Windows窗口
- 使用Pillow和pytesseract尝试从计算器界面读取结果

## 安装依赖

```bash
pip install -r requirements.txt
```

注意：如果需要OCR功能读取计算器结果，还需要安装[Tesseract-OCR](https://github.com/UB-Mannheim/tesseract/wiki)。

## 运行应用

```bash
python ai_calculator.py
```

## 使用方法

1. 在输入框中输入数学表达式（如：5+3）
2. 点击"计算 (=)"按钮
3. 应用会自动启动系统计算器并输入表达式
4. 系统计算器计算完成后，结果会显示在本应用界面
